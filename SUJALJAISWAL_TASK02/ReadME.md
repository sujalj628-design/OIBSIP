# OIBSIP : DATA SCIENCE INTERNSHIP
## TASK02 : SALES PREDICTION USING PYTHON

This project predicts product sales based on advertising expenditures on different media platforms such as TV, Radio, and Newspaper using Machine Learning.The objective of this project is to predict sales by analyzing advertising budgets spent on TV, Radio, and Newspaper advertisements.

The Advertising dataset contains advertising budgets and corresponding sales data
* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget

Technology used in making this project are
* Python
* Pandas
* Scikit-learn

Algorithm used in making this model is Linear Regression.

The workflow of the project-
1. Load the Advertising dataset
2. Explore and understand the dataset
3. Select input features and target variable
4. Split the dataset into training and testing sets
5. Train the Linear Regression model
6. Evaluate model performance using R² Score
7. Predict sales based on user input

The user can enter:
* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
The model predicts the expected sales value.
  
Conclusion- This project demonstrates how Machine Learning can be used to predict sales based on advertising investments. The Linear Regression model successfully identifies the relationship between advertising expenditure and sales.
