## TASK01 : IRIS FLOWER CLASSIFICATION

 This project is a machine learning model that classify Iris flower into different species based on their speal and petal measure. The main objective is to predict the species using machine learning.
 
 The Iris dataset contains 150 flower sample which have sepal length, sepal width, petal length and petal width. Iris have 3 specificaton that are 
 = Iris-setosa
 = Iris-versicolor
 = Iris-virginica.

 Technology used in making this project are
 = Python
 = Pandas
 = Scikit-learn

 Algorithm used in making this model is Random Forest Classifier.

 The workflow of the project-
 1. Load the Iris dataset
 2. Explore and understand the dataset
 3. Select features
 4. Split data into training and testing sets
 5. Train the Random Forest model
 6. Evaluate model performance
 7. Predict flower species based on user input


 In this model users can enter:
 Sepal Length
 Sepal Width
 Petal Length
 Petal Width
 Then the model predicts the corresponding Iris flower species.

 Conclusion- The Random Forest Classifier successfully classified Iris flowers with high accuracy (100%) and demonstrated the effectiveness of Machine Learning for classification tasks to train a machine learning model that can learn from the
 measurements of the iris species and classify them.
